Two equal traveling waves are added point by point. Resultant amplitude = 2·cos(Δφ/2); intensity ∝ amplitude², so it ranges from 4× a single wave (in phase) down to 0 (Δφ = π). Path difference is just Δφ expressed in wavelengths.

Live wavefronts from two in-phase sources. Each photon is fired one at a time and lands at a random point drawn from the diffraction probability distribution. No single photon shows a pattern, but the histogram of many independent detections converges to the interference fringes. Block one slit and the fringes vanish, leaving the broad single-slit envelope, because there is now only one path and nothing to interfere with.
